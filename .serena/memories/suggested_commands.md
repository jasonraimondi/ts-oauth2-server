# Suggested commands

- Install deps: `pnpm install`
- Build: `pnpm build`
- Test all: `pnpm test`
- Watch tests: `pnpm test:watch`
- Coverage: `pnpm test:cov`
- Format: `pnpm format`
- Single test file: `pnpm vitest run path/to/file.spec.ts`
- Dev/watch build: `pnpm start` (`tsc -p tsconfig.build.json --watch`)
- Useful Darwin shell tools: `git status`, `git diff`, `ls`, `find`, `rg`, `grep`.
